Java使用jdbc连接mysql数据库

连接时需要注意的几点： 1. 将jdbc的jar包放入项目文件lib中 2. 记住mysql安装时的用户名和密码 3. 尽量将类文件放在util目录下 4. 在本地使用时一般只需要改数据库user和pass还有databasename 5. 在其他类中调用完毕后记得关闭连接 6. 新建一个文件名为DBUtil再将代码放入即可
