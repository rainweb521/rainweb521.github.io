JavaWeb—Blog code

本项目使用JavWeb编写，结构层次分明，使用Action,Factory，DAOImpl分级传递，前台功能基本完善，有顶部菜单和侧栏菜单底部菜单，支持首页展示功能，文章功能，文章包括列表显示和单篇文章显示功能。相册功能，可以按时间和上传者分别展示，留言版功能，支持留言和已有留言的游览，数据库db_blog.sql文件已经放到database文件夹中，导包即可使用
